
Projeto criado para análise de conhecimento para Millenium Brasil.
Nesse projeto foi utilizado .NET, C#, Visual Studio 2017, Linq, 
padrões de projeto DDD com princípios SOLID,
múltiplas camadas de projeto, acesso a banco de dados através de EntityFramework, 
banco de dados SQL SERVER, Migrations entre outros.

Projeto publicado no GitHub e poderá ser acessado através do link abaixo:
https://github.com/Viniciustes/GamesParseLog

Artefatos de criação de banco está na pasta ScriptsDB

Utilizado banco local, para rodar o projeto tem que mudar a connection string.
